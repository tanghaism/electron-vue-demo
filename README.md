electron使用vue的样板

```bash
# Install dependencies
npm install --unsafe-perm=true --allow-root

# Run in `DEV` mode
npm run dev

# Build installer for this app
npm run build

# Build directory for this app with executeable
npm run build:dir

# Lint all source files using ESLINT
npm run lint
```
